# How to run program
To run the to-do list app, you are presented with a menu of 5 options. 
You are allowed to choose which action you want and enter that number. Then follow the prompts given to complete that action. 

# MVP
Actions that can be done in the mvp implematation:
1. Add a task
2. Remove a task
3. View all tasks
4. Mark a task as Complete
